# Projet Fractal Three.js

Ce projet est une **version 3D d’un fractal**, inspirée d’une première version en 2D. Il utilise **Three.js** pour générer des formes fractales.
Je ne suis pas totalement satisfait du résultat, car ce n’est pas vraiment aléatoire. Les cubes ne font que se placer les uns à la suite des autres :/

---


## Installation

Assurez-vous d’avoir **Node.js** et **npm** installés.

1. Installer les dépendances

```bash
npm install
```

---

## Lancer le projet

Pour construire et lancer le projet en développement :

```bash
npm run build    # Compile le projet
npm run dev      # Lancer le serveur de développement
```
